顺序表（挺简单，直接上代码）

  • InitList(&L) : 初始化顺序表
  • DestroyList(&L) : 销毁顺序表
  • ListEmpty(L) : 判断顺序表是否为空
  • ListLength(L) : 获取顺序表中当前元素的数量，即线性表长度
  • DisplayList(L) : 输出顺序表
  • GetElem(L, i, &e) : 获取顺序表的第i个元素
  • LocateElem(L, e) : 查找并返回顺序表中第一个值为e的元素的序号
  • ListInsert(&L, i, e) : 在顺序表的第i个位置插入元素e
  • ListDelete(&L, i, &e) : 删除顺序表的第i个元素
